# Primer Proyecto Angular

¡Bienvenido/a a mi primer proyecto con Angular!  
Este repositorio es parte de mi aprendizaje de Angular y contiene la base para futuras aplicaciones, experimentos y mejoras que realice mientras aprendo el framework.

---

## 🛠 Tecnologías

- **Angular**
- **TypeScript**
- **SCSS**
- **HTML**

---

## 🚀 Instalación y Ejecución

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/aperezm1/Primer-Proyecto-Angular.git
   cd Primer-Proyecto-Angular
   ```

2. **Instala las dependencias**
   ```bash
   npm install
   ```

3. **Ejecuta en modo desarrollo**
   ```bash
   ng serve
   ```
   Luego abre [http://localhost:4200](http://localhost:4200) en tu navegador.
